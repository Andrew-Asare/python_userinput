# User Input 
## Task
- The task is to get the user to enter their FirstName, LastName, Age, Salary, DOB and Post Code

## How to complete

- To complete this task we need to put an input to each data type and save it under a variable

-Here is an example of using an input to get the user to fill out the information
```python
last_name = input("What is your last name?\n")
```
- To show the output of this we would use a print statment as such:

```python
print(last_name)
```
